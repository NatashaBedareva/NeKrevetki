# 🦐 Проект «НеКреветка»

Мобильное приложение для контроля осанки в реальном времени с использованием компьютерного зрения и ИИ.

## 📋 О проекте

**НеКреветка** — это приложение, которое помогает следить за осанкой во время работы за компьютером. Оно использует камеру смартфона для анализа положения тела в реальном времени и предупреждает пользователя, когда он начинает сутулиться.

### Основные возможности

- 🎥 **Анализ видеопотока в реальном времени** (до 30 кадров/сек)
- 🧠 **Детекция скелета человека** с помощью нейросети
- 📐 **Расчёт углов наклона** плеч, шеи и позвоночника
- 🔧 **Калибровка под пользователя** (запоминание идеальной позы)
- 🔔 **Звуковые и визуальные уведомления** при сутулости
- 📊 **Статистика нарушений** для отслеживания прогресса

### Проблема, которую решает проект

Миллионы людей страдают от болей в спине из-за сутулости во время сидячей работы. Самостоятельно следить за осанкой сложно — мозг фокусируется на работе, а не на положении тела.

**«НеКреветка»** действует как персональный тренер: постоянно наблюдает и напоминает: *«Сядь прямо!»*
